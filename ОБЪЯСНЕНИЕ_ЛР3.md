## Основні допоміжні функції

### 1. `generateRandomArray(size, min, max)`
```javascript
function generateRandomArray(size, min, max) {
    const array = [];
    for (let i = 0; i < size; i++) {
        array.push(Math.floor(Math.random() * (max - min + 1)) + min);
    }
    return array;
}
```
**Призначення:** Генерація одновимірного масиву випадкових чисел
- **Параметри:** розмір масиву, мінімальне та максимальне значення
- **Алгоритм:** `Math.random() * (max - min + 1) + min` - стандартна формула для отримання випадкового числа в діапазоні
- **Повертає:** Масив цілих чисел заданого розміру

### 2. `generateRandom2DArray(rows, cols, min, max)`
```javascript
function generateRandom2DArray(rows, cols, min, max) {
    const array = [];
    for (let i = 0; i < rows; i++) {
        const row = [];
        for (let j = 0; j < cols; j++) {
            row.push(Math.floor(Math.random() * (max - min + 1)) + min);
        }
        array.push(row);
    }
    return array;
}
```
**Призначення:** Генерація двовимірного масиву випадкових чисел
- **Структура:** Масив масивів (rows × cols)
- **Вкладені цикли:** Зовнішній для рядків, внутрішній для стовпців
- **Використовує:** принцип `generateRandomArray` для кожного елемента

### 3. `createTableFromArray(array)`
```javascript
function createTableFromArray(array) {
    const table = document.createElement('table');
    array.forEach(row => {
        const tr = document.createElement('tr');
        row.forEach(cell => {
            const td = document.createElement('td');
            td.textContent = cell;
            tr.appendChild(td);
        });
        table.appendChild(tr);
    });
    return table;
}
```
**Призначення:** Створення HTML таблиці з двовимірного масиву
- **DOM маніпуляції:** Динамічне створення елементів table, tr, td
- **Вкладена структура:** forEach для рядків та стовпців
- **Візуалізація:** Зручне відображення двовимірних даних

## Завдання 2: Одновимірні масиви

### 2.1. `find_min_even()` - Знайти найменший парний елемент або перший
```javascript
function find_min_even() {
    const num = parseFloat(document.getElementById('num_one_arr_2_1').value);
    const array = generateRandomArray(num, -100, 0);
    
    let minEven = null;
    for (let i = 0; i < array.length; i++) {
        if (array[i] % 2 === 0) {          // Проверка на четность
            if (minEven === null || array[i] < minEven) {
                minEven = array[i];
            }
        }
    }
    
    const result = minEven !== null ? minEven : array[0];
    const message = minEven !== null 
        ? `Найменший парний елемент: ${result}` 
        : `Парних елементів немає. Перший елемент: ${result}`;
}
```
**Алгоритм:**
1. **Пошук парних чисел:** `array[i] % 2 === 0` - перевірка залишку від ділення на 2
2. **Пошук мінімуму серед парних:** Порівняння з поточним мінімумом
3. **Умовна логіка:** Якщо парних немає → повертається перший елемент
4. **Ініціалізація:** `minEven = null` для визначення першого парного

**Особливість:** Працює з від'ємними числами (парні від'ємні: -100, -98, -96...)

### 2.2. `find_min_max_elements()` - Знайти мінімум та максимум
```javascript
function find_min_max_elements() {
    const array = generateRandomArray(num, -100, 0);
    
    let minValue = array[0];
    let maxValue = array[0];
    
    for (let i = 1; i < array.length; i++) {     // Начинаем с индекса 1
        if (array[i] < minValue) {
            minValue = array[i];
        }
        if (array[i] > maxValue) {
            maxValue = array[i];
        }
    }
}
```
**Алгоритм пошуку екстремумів:**
1. **Ініціалізація:** Перший елемент як мінімум та максимум
2. **Лінійний пошук:** Один прохід по масиву (O(n))
3. **Подвійне порівняння:** На кожній ітерації перевіряємо min та max
4. **Оптимізація:** Починаємо з індексу 1 (не 0)

### 2.3. `calculate_index_sums()` - Сума індексів парних/непарних елементів
```javascript
function calculate_index_sums() {
    const array = generateRandomArray(num, -100, 0);
    
    let evenIndexSum = 0;      // Сумма индексов четных элементов
    let oddIndexSum = 0;       // Сумма индексов нечетных элементов
    
    for (let i = 0; i < array.length; i++) {
        if (array[i] % 2 === 0) {
            evenIndexSum += i;       // Прибавляем ИНДЕКС, не значение!
        } else {
            oddIndexSum += i;
        }
    }
}
```
**Важлива відмінність:** Сумуються **індекси** елементів, а не їх значення
- `array[i] % 2 === 0` - перевіряємо ЗНАЧЕННЯ елемента
- `evenIndexSum += i` - додаємо ІНДЕКС елемента
- **Приклад:** array = [-10, -7, -8], парні значення: -10 (індекс 0), -8 (індекс 2) → сума індексів = 0 + 2 = 2

## Завдання 3: Двовимірні масиви

### 3.1. `count_positive_elements()` - Підрахунок позитивних елементів
```javascript
function count_positive_elements() {
    const array = generateRandom2DArray(rows, cols, -100, 100);  // Диапазон [-100, 100]
    
    let count = 0;
    for (let i = 0; i < array.length; i++) {          // Перебор строк
        for (let j = 0; j < array[0].length; j++) {    // Перебор столбцов
            if (array[i][j] > 0) {                     // Проверка на положительность
                count++;
            }
        }
    }
}
```
**Алгоритм двовимірного пошуку:**
1. **Вкладені цикли:** `i` для рядків, `j` для стовпців
2. **Умова:** `array[i][j] > 0` (суворо більше нуля)
3. **Лічильник:** Інкремент при знайденому позитивному елементі
4. **Розмірність:** `array[0].length` - кількість стовпців у першому рядку

**Особливість:** Діапазон змінився на [-100, 100] для наявності позитивних чисел

### 3.2. `find_min_element()` - Пошук мінімального елемента в 2D масиві
```javascript
function find_min_element() {
    const array = generateRandom2DArray(rows, cols, -100, 0);    // Обратно к [-100, 0]
    
    let minElement = array[0][0];                    // Инициализация первым элементом
    for (let i = 0; i < array.length; i++) {
        for (let j = 0; j < array[0].length; j++) {
            if (array[i][j] < minElement) {
                minElement = array[i][j];
            }
        }
    }
}
```
**Алгоритм пошуку мінімуму в 2D:**
1. **Ініціалізація:** `array[0][0]` - перший елемент матриці
2. **Повний перебір:** Кожен елемент матриці порівнюється з поточним мінімумом
3. **Часова складність:** O(rows × cols)
4. **Альтернатива:** Можна було б порівнювати порядково, але прямий підхід зрозуміліший

## Математичні концепції

### 1. Парність чисел
```javascript
array[i] % 2 === 0  // Парне число (залишок 0)
array[i] % 2 !== 0  // Непарне число (залишок 1 або -1)
```
**Особливість з від'ємними числами:**
- `-4 % 2 === 0` ✓ (парне)
- `-3 % 2 === -1` ✗ (непарне, але залишок -1, не 1)

### 2. Пошук екстремумів
**Лінійний алгоритм:** O(n) для одновимірних, O(n×m) для двовимірних
**Ініціалізація:** Першим елементом, не `+Infinity/-Infinity`

### 3. Генерація випадкових чисел
```javascript
Math.floor(Math.random() * (max - min + 1)) + min
```
**Розбір формули:**
- `Math.random()` → [0, 1)
- `Math.random() * (max - min + 1)` → [0, max-min+1)  
- `Math.floor(...)` → [0, max-min] (цілі)
- `+ min` → [min, max] (зсув діапазону)

// Єдиний прохід для пошуку min, max та парних
function findMultipleStats(array) {
    let min = array[0], max = array[0], minEven = null;
    for (let i = 0; i < array.length; i++) {
        if (array[i] < min) min = array[i];
